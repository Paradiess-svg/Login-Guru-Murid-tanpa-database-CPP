   <b>CATATAN, PROGRAM INI SUDAH SELESAI. PROGRAM BEBAS DIPERGUNAKAN SECARA PUBLIK KHUSUSNYA PENDIDIKAN</b>

User: Admin

- Menghitung rata-rata nilai untuk setiap siswa
